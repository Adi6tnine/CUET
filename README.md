# AVION.EXE

A personal command center for academic and skill development with Linux Fundamentals integration.

## Features

- **Skill War Room**: Linux Fundamentals, Bash Scripting, Python Automation, AI Security
- **Academic Tracking**: Syllabus progress and timeline management
- **Daily Protocol**: Routine enforcement and consistency tracking
- **Smart Analytics**: Performance insights and progress analysis
- **Zero Data Loss**: Local-first with sync capabilities

## Quick Start

1. Clone the repository
2. Install dependencies: `npm install`
3. Set up environment variables (copy `.env.example` to `.env`)
4. Run development server: `npm run dev`
5. Build for production: `npm run build`

## Environment Variables

```
GROQ_API_KEY=your_groq_api_key_here
```

## Deployment

This app is optimized for Vercel deployment with serverless functions for AI features.

## Tech Stack

- React + Vite
- Framer Motion
- Tailwind CSS
- IndexedDB
